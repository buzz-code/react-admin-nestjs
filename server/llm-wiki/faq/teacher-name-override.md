## שאלה נפוצה
"רוצים שנחליף את שם המורה במקצוע תפילה שיהיה '-----' (או שם אחר)."

## הסבר
יש כפתור "שינוי מורה" בטבלת הנוכחות (att_report_with_report_month) — פעולה קבוצתית שמעדכנת את שדה `teacherReferenceId` ברשומות הנוכחות.

**חשוב:** הכפתור דורש בחירת מורה **קיים** מתוך רשימת המורות/מורים במערכת — אין אפשרות להקליד טקסט חופשי. אם לא נבחר מורה, השרת דוחה את הבקשה ("לא נבחר מורה").

כלומר, כדי שיוצג "-----" (או כל שם אחר) בתור שם המורה, חייבת להיות במערכת רשומת מורה קיימת בשם הזה מראש.

## תשובה מוצעת
אתם יכולים לעשות את זה דרך הכפתור "שינוי מורה" בטבלת נוכחות. שימו לב — הכפתור דורש לבחור מורה קיים מהרשימה, אין אפשרות טקסט חופשי. כמובן שצריך לוודא שיש מורה בשם הזה (למשל "-----") מוגדר מראש במערכת, ולבחור אותו מהרשימה.
